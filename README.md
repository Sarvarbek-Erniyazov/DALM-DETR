# OffsetIoU-Det

**Location-Aware Matching for Dense & Occluded Object Detection**

A query-based, NMS-free detector that augments the Hungarian matching cost
with a location-aware term to reduce ambiguous assignments in dense and
occluded scenes. Evaluated on CrowdHuman under a controlled, honest ablation.
